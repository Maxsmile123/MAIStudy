# Курсовой проект. Задание 8. Линейные списки  
Сделать на указателях  
Тип элемента: литерный  
Вид списка: линейный, однонаправленный с барьерным элементом  
Нестандартное действие: переставить элементы списка в обратном порядке  
Реализовано на итераторах
# Как запускать?  
Компилируем все .c файлы и запускаем через ./a.out  
Взаимодейсвтие с пользователем интерактивное, с выводом списка опций

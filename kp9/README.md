# Курсовой проект. Задание 9. Сортировка и поиск  
Вариант 3-1.  
Сортировка методом пузрька  
Хранение таблицы - № 1.  
# Структура проекта
main.c - файл с реализацией интерактивного меню   
table.h - заголовочный файл для некоторых функций, связанных с таблицами   
table.c - реализация того, что в table.h   
func.h - заголовочный файл со всеми нужными функциями - бинарный поиск, сортировка, реверс, перемешивание   
func.c - реализация func.h
# Как запускать?
В файл input.txt необходимо занести данные в виде: целое число строка   
Целое число - ключ типа long.   
Строка максимум в 120 символов   
А далее, по классике - gcc + ./a.out

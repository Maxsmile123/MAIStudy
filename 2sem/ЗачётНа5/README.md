# Зачёт на оценку 5 по дисциплине "Практикум на ЭВМ"  
Задание:   
Реализовать скрипт выполняющий сортировку строк методом двоичной кучи (пирамидальная сортировка).
Интерфейс должен соответствовать аналогичной утилите командной строки sort, то есть при наличии данных во входном потоке следует
использовать их, иначе - читать из файла указанного в аргументах скрипта.

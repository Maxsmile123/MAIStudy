# Лабораторная работа 23. Динамические структуры данных. Обработка деревьев  

Вариант 1.  
Проверить, является ли двоичное дерево AVL-деревом  
В программе реализованы следующие методы:  
1) Добавление нового элемента в дерево

4) Проверка дерева на AVL-дерево

6) Поиск максимального элемента дерева

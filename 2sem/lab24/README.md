# Лабораторная работа №24. Дерево выражений  
Вариант 32  
Подсчитать количество слагаемых или сомножителей на k-ом уровне дерева выражений  
# Структура проекта
tree.h - заголовочный файл с функциями для дерева      
tree.c - файл с реализацией функций из tree.h   
main.c - файл с реализацией ввода выражения, построения выражения и взаимодействия с пользователем   
# Как запускать?  
gcc main.c tree.c  
./a.out

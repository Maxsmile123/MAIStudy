Лабораторная работа №23  
Множества  
Вариант 7  
):  На вход подаётся произвольный набор английских слов, разделённых пробелами, запятыми, знаками табуляции и границами строк. Букву 'y' для простоты считать согласной (гласными или согласными бывают звуки, но не буквы). Необходимо проверить выполнение следующего условия: есть ли слова, начинающиеся и заканчивающиеся гласными? При решении задачи необходимо реализовать математическую абстракцию множества (сам тип, полное/пустое множество, пересечение, симметрическая разность и т.д.). Для реализации множества необходимо использовать битовые операции, но при этом запрещается нарушать принципы абстракции и инкапсуляции: пользовательский код должен зависеть только от интерфейса, но не от его реализации. При имплементации множеств можно считать, что все они имеют мощность не больше 32. Помимо обычных заголовочных файлов разрешается использовать ctype.h, inttypes.h и stdbool.h. Код функции main должен зависеть только от интерфейса АТД множество.

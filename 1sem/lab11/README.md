# Лабораторная работа №11. Конечные автоматы и регулярные выражения  
Вариант 26  
Составить программу, выполняющую анализ и обработку вводимого ascii-текста. Требуется построчно выделить предпоследнее десятичное число и вычислить сумму его цифр. При решении задачи необходимо использовать матаппарат конечных автоматов. Запрещено использовать массивы, указатели и явные числовые коды ascii-знаков (например, число 48 вместо литерной константы '0').

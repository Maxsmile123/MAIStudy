# Лабораторная работа №15. Обработка матриц  
Вариант 3  
На вход подаётся пакет тестов, содержащий набор квадратных матриц. Каждую матрицу, поданную на вход, необходимо обработать согласно заданию: перестановка столбцов с максимальной и минимальной суммой элементов (с минимальными номерами соответственно).
